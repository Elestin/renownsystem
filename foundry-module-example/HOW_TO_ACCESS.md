# How to Access Renown System in Foundry VTT

## Method 1: Admin Configuration Panel (GM Only)

**For setting up factions, relationships, and themes:**

1. Click the **⚙️ Settings** button (gear icon) in the top menu bar
2. Go to **Module Settings**
3. Find **"Renown System"** in the list
4. Click **"Configure World"** button
5. The admin panel will open!

**Visual Guide:**
```
Top Menu Bar
  ↓
⚙️ Settings → Module Settings → Renown System → Configure World
```

## Method 2: Renown Tracker (Scene Controls)

**For tracking renown values during gameplay:**

The Renown Tracker button should appear in the **left sidebar** (Scene Controls). Look for:

1. On the **left side of the screen**, you'll see a vertical toolbar with icons
2. This is the "Scene Controls" panel
3. Look for a **⭐ star icon** button
4. Click it to open the Renown Tracker

**If you don't see the star icon:**

- Make sure you're logged in as a **GM** (Game Master)
- The button only appears for GMs
- Try refreshing the page (F5)
- Check the browser console (F12) for any errors

**Visual Guide:**
```
Left Sidebar (Scene Controls)
  ↓
[Other icons...]
[⭐ Renown Tracker] ← Click this!
[Other icons...]
```

## Method 3: Alternative - Add to Hotbar (Optional)

If you want easier access, you can:

1. Open the Renown Tracker once (via Scene Controls)
2. Right-click on the window title bar
3. Select "Pin to Hotbar" (if available)
4. Or drag it to your hotbar at the bottom

## Troubleshooting

### "I don't see the star icon in Scene Controls"

**Check:**
1. Are you logged in as GM? (Check top right - should say "Gamemaster")
2. Is the module enabled? (Setup → Manage Modules → Renown System should be checked)
3. Open browser console (F12) and look for errors
4. Try reloading the page (F5)

### "I can't find Module Settings"

1. Click the **⚙️ gear icon** in the top menu
2. Look for **"Module Settings"** (not "Configure Settings")
3. It should be in the list on the left side

### "The Configure World button doesn't work"

1. Make sure you're a GM
2. Check browser console (F12) for errors
3. Try disabling and re-enabling the module

## Quick Access Tips

- **Admin Panel**: Settings → Module Settings → Renown System → Configure World
- **Tracker**: Left sidebar → ⭐ Star icon
- Both require GM permissions

## Still Having Issues?

1. Press **F12** to open browser console
2. Look for any red error messages
3. Take a screenshot and check:
   - Is the module enabled?
   - Are you logged in as GM?
   - Are there any JavaScript errors?
